Binary = Incremented by 2
Decimal = Incremented by 10

#######################

Conversion functions:

bin() = binary conversion
int() = decimal conversion
hex() = hexidecimal conversion

example:

>>> bin(25)
'0b11001'
>>> hex(0b11001)
'0x19'
>>> int(0x19)
25

Math Functions

import math	imports math library
math.sqrt()	square root
math.exp()	exponential or "to the power of"
math.log()	the natural (base e) logarithm of variable
math.ceil()	the smallest whole number of variable
math.floor()	the largest whole number of variable
math.abs()	absolute value
math.max(x,y)	maximum value
math.min(x,y)	minimum value
